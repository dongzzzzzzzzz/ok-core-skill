# 看房决策清单

实地看房时的系统化检查指南。Agent 在推荐房源进入"看房阶段"时，向用户提供对应的检查项。

---

## 一、噪音评估

噪音是影响居住体验的首要因素，且几乎无法通过装修改善。

### 5 类噪音源

| 来源 | 识别方法 | 影响程度 | 线上预判工具 |
|------|---------|---------|------------|
| **MRT/铁路轨道** | 公寓靠近高架轨道段 | 高 — 每 2-5 分钟一次 | Google Maps 卫星图查看轨道位置 |
| **主干道/高速** | 低楼层面向主干道 | 高 — 持续性噪音 | Google Maps Street View 查路况 |
| **建筑工地** | 周边有在建项目 | 高 — 持续 2-3 年 | 当地规划网站（URA/Municipality）查开发项目 |
| **中庭回音** | 大型公寓（500+ 户）中庭户型 | 中 — 泳池/儿童区活动噪音 | Google Maps 卫星图看公寓布局 |
| **空调外机** | 邻居/楼下外机靠近卧室窗 | 中 — 夜间更明显 | 只能实地确认 |

### 实测方法

1. **关窗测试**：关闭所有窗户，安静站 5 分钟，感受基础噪音
2. **开窗测试**：打开所有窗户，站 5 分钟，对比差异
3. **夜间复访**：晚上 9-10 点再来一次，夜间噪音更明显（空调、酒吧、货车等）
4. **量化工具**：手机下载 Sound Meter 或 Decibel X app

### 噪音阈值参考

| 分贝 (dB) | 相当于 | 居住评价 |
|-----------|--------|---------|
| < 30 | 耳语 | 极安静，理想 |
| 30-40 | 图书馆 | 安静，优良 |
| 40-50 | 正常交谈 | 可接受 |
| 50-60 | 办公室 | 需要关窗，勉强 |
| > 60 | 繁忙马路 | 不建议，长期影响健康 |

## 二、朝向与采光

### 朝向评估

| 朝向 | 评价 | 说明 |
|------|------|------|
| **南北通透** | 最佳 | 采光好，通风佳，无西晒 |
| **东向** | 良好 | 早晨阳光，下午不晒 |
| **北向** | 一般 | 采光偏弱，但无晒 |
| **西向** | 需注意 | 下午西晒严重，空调费增加 20-30% |

### 采光检查

- [ ] 下午 2-4 点去看（能准确判断西晒）
- [ ] 确认周边是否有高楼遮挡采光
- [ ] 低楼层（1-5 楼）注意树木/建筑遮挡
- [ ] 确认卧室窗户朝向（比客厅朝向更重要）

### 楼层选择

| 楼层 | 优势 | 劣势 |
|------|------|------|
| 低层 (1-5) | 等电梯快、逃生方便 | 采光差、蚊虫多、隐私低 |
| 中层 (6-15) | 平衡之选 | 无特别劣势 |
| 高层 (16-30) | 采光好、视野佳、隐私好 | 等电梯久、风大、搬家成本高 |
| 超高层 (30+) | 景观极佳 | 电梯高峰等 5-10 分钟、水压可能不足 |

## 三、实用设施检查

### 室内设施

- [ ] **洗衣晾晒**：是否有 yard/utility room？公寓是否禁止外晒？
- [ ] **网络**：确认宽带供应商选择（部分公寓有独家合约，无法选运营商）
- [ ] **空调类型**：中央空调（无法独立控温）vs 分体空调（更灵活）
- [ ] **热水器**：即热式 vs 储水式，影响洗澡体验
- [ ] **厨房**：燃气灶 vs 电磁炉，抽油烟机功率是否足够（中式烹饪重要）
- [ ] **储物空间**：衣柜深度、储藏间面积

### 公共设施

- [ ] **电梯数量**：30 层以上公寓，<= 2 部电梯意味着高峰等待 5+ 分钟
- [ ] **停车位**：含在租金内还是额外收费？访客停车方便吗？
- [ ] **门禁系统**：是否影响外卖/快递收取？
- [ ] **垃圾处理**：垃圾房位置（靠近单位可能有异味）
- [ ] **泳池/健身房**：状态如何？拥挤度？

### 水电检查

- [ ] 打开所有水龙头测水压（同时开 2-3 个）
- [ ] 冲厕所测排水速度
- [ ] 检查墙角、天花板有无水渍/霉斑（漏水信号）
- [ ] 每个房间开关灯，检查电路

## 四、费用相关

### 必须确认的费用项

| 费用项 | 确认要点 | 常见陷阱 |
|--------|---------|---------|
| **管理费** | 谁付？金额多少？ | 高管理费（$400+/月）意味着房东更易涨租转嫁 |
| **水电费** | 是否包含在租金内？ | 部分高端公寓含中央空调费（chiller），不含则额外支出大 |
| **空调清洗** | 维修责任方？ | 热带地区建议每季度清洗，费用 $80-150/次 |
| **网络费** | 是否有独家合约限制？ | 独家合约可能价格高且无法更换 |
| **垃圾费** | 部分地区需单独付 | 日本、部分欧洲国家有垃圾处理费 |

### 入住成本速算

```
月度总成本 = 租金 + 水电预估 + 网络 + 停车费 + 空调维护分摊
入住总成本 = 押金 + 首月租金 + 中介费(如有) + 印花税(如有) + 搬家费
```

## 五、租约条款

### 必须逐条确认

- [ ] **租期**：最短几个月？续租条件？
- [ ] **提前解约**：是否允许？违约金多少？（通常 1-2 个月租金 + 剩余租期赔偿）
- [ ] **续租涨幅**：是否有涨幅上限条款？（如"涨幅不超过 10%"）
- [ ] **押金退还**：什么情况扣？折旧标准如何？入住时拍照存证！
- [ ] **维修责任**：什么归房东修？什么归租客修？（通常大件归房东，消耗品归租客）
- [ ] **转租**：是否允许转租？（如果有提前离开的可能性，非常重要）
- [ ] **宠物政策**：是否允许？额外押金？
- [ ] **入住时间**：确认前任租客搬出时间，避免空等

### 入住前必做

1. **全屋拍照/录像**：拍摄每个房间、每面墙、所有设备状态，作为退租时的证据
2. **记录已有损坏**：划痕、水渍、设备故障，书面告知房东并保存记录
3. **抄表读数**：水表、电表入住时的读数

| 工具 | 用途 |
|------|------|

| 指南针 (手机自带) | 确认朝向 |

| 充电宝 | 看房一天手机容易没电 |
